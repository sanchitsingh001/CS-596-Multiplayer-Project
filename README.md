# Multiplayer Project — CS596 Spring 2026

Assignment: multiplayer Tic-Tac-Toe using Unity and Netcode for GameObjects (Host/Client, NetworkVariables, RPCs).

## Testing multiplayer

As shown in the video: use **one build** (File > Build and Run) and **one instance in the Unity Editor**. Treat them as two separate devices—they can connect and talk to each other.

1. Run the **built game** → click **Host**.
2. In the **Editor**, press Play → click **Client** (leave Server Address as 127.0.0.1, Port 7778).
3. Play a game; moves and scores sync between the two.

Before first run: in Unity use **TicTacToe > Setup Multiplayer Scene**, then press Play.
